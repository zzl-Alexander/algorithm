Acwing 68. 0到n-1中缺失的数字

5月 03 2021

一个长度为 n−1 的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围 0 到 n−1 之内。

在范围 0 到 n−1 的 n 个数字中有且只有一个数字不在该数组中，请找出这个数字。

思路

假设值为x的数缺失了
因为数组本来时递增排序的
下表：0 1 2 3 4 … x-1 x x+1…
值：0 1 2 3 4 … x x+1 x+2…
发现x位置之前的数都满足nums[i]=i，直接进行二分
复杂度o(log(n))

直接暴力跑一遍是复杂度o(n)

代码

二分
